# Documentation catalog

Read this first. Every doc for the NREL Lab showcase site is listed here.

An interactive version of this catalog — with links to the live pages — is `docs/index.html`
(open it in a browser). The markdown files below render nicely via `docs/_view.html?p=<file>`
when the folder is served over a local HTTP server.

## One-line orientation

- The site is **static HTML/CSS/JS**, 7 pages, no build step.
- It matches the **REAL Lab brand** (cream + navy + gold), light theme.
- All data is a **public July 2026 snapshot** (Google Scholar + the lab's homepage).
- The **home hero citation chart** is the signature — keep it.
